快速开始

5 分钟配置好开发工具链

1

安装开发工具

Bashcomposer require --dev \ laravel/pint \ phpstan/phpstan \ phpstan/phpstan-laravel \ nunomaduro/larastan \ pestphp/pest \ pestphp/pest-plugin-laravel
2

运行代码格式化

Bash./vendor/bin/pint # 自动格式化 ./vendor/bin/pint --test # 检查但不修改
3

运行静态分析

Bash./vendor/bin/phpstan analyse # Level 9 静态分析
4

运行测试

Bashphp artisan test # 运行所有测试 php artisan test --coverage # 带覆盖率报告
