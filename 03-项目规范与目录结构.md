## 📝 命名规范

### 类和接口
```php
// ✅ 使用 PascalCase
class UserController { }
class UserRepository { }
class UserService { }
class RegisterUserAction { }
class UserRegistered { } // 事件
class UserNotFoundException { } // 异常

// ❌ 错误示例
class user_controller { }
class UserRepositories { } // 避免复数
class get_user_service { }
```

### 方法和函数
```php
// ✅ 使用 camelCase
public function getUserById(int $id) { }
public function createUser(array $data) { }
public function isEmailUnique(string $email) { }
public function handle() { }

// ❌ 错误示例
public function GetUserById(int $id) { }
public function get_user_by_id(int $id) { }
public function getUserByID(int $id) { } // ID 不拆分
```

### 常量
```php
// ✅ 使用 UPPER_SNAKE_CASE
const MAX_RETRY_COUNT = 3;
const DEFAULT_PAGE_SIZE = 20;
const CACHE_TTL_HOURS = 24;

// ❌ 错误示例
const maxRetryCount = 3;
const max_retry_count = 3;
```

### 变量
```php
// ✅ 清晰的名称
$activeUsers = User::active()->get();
$userCount = $users->count();
$isAdmin = $user->hasRole('admin');

// ❌ 模糊的名称
$users = User::active()->get();  // 是什么 users？
$count = $users->count();
$admin = $user->hasRole('admin');  // 返回什么？
```

## 🗂️ 文件放置原则

### Models（模型）
```php
// app/Models/User.php
class User extends Model
{
    // 数据实体，定义关联关系
    public function profile() { }
    public function orders() { }
}
```

### Repositories（仓储）
```php
// app/Repositories/UserRepository.php
class UserRepository
{
    // 所有数据查询逻辑
    public function getActive() { }
    public function findById(int $id) { }
    public function create(array $data) { }
}
```

### Services（服务）
```php
// app/Services/UserService.php
class UserService
{
    // 业务逻辑和流程编排
    public function activateUser(int $userId) { }
    public function sendWelcomeEmail(User $user) { }
}
```

### Actions（操作）
```php
// app/Actions/RegisterUserAction.php
class RegisterUserAction
{
    // 复杂的多步骤操作
    public function execute(array $data): User { }
}
```

### Controllers（控制器）
```php
// app/Http/Controllers/UserController.php
class UserController extends Controller
{
    // 只负责请求/响应处理，调用 Service 或 Action
    public function store(StoreUserRequest $request, RegisterUserAction $action)
    {
        $user = $action->execute($request->validated());
        return UserResource::make($user);
    }
}
```

## 🔧 编码风格指南

### 1. 类型提示完整
```php
// ✅ GOOD
public function getUsersByRole(string $role, int $limit = 10): Collection
{
    return $this->repository->getByRole($role, $limit);
}

// ❌ BAD
public function getUsersByRole($role, $limit = 10)
{
    return $this->repository->getByRole($role, $limit);
}
```

### 2. 使用命名参数
```php
// ✅ GOOD - 清晰的参数含义
User::paginate(
    perPage: 20,
    columns: ['id', 'name', 'email'],
    pageName: 'page'
);

// ❌ BAD - 需要查看方法签名
User::paginate(20, ['id', 'name', 'email'], 'page');
```

### 3. 链式调用
```php
// ✅ GOOD - 链式调用提高可读性
$users = User::query()
    ->where('active', true)
    ->orderByDesc('created_at')
    ->paginate();

// ❌ BAD - 过度使用
$users = User::where('active', true)->orderByDesc('created_at')->paginate();
```

### 4. 使用现代 PHP 特性
```php
// ✅ GOOD - 使用 readonly 和构造器升级
final class UserService
{
    public function __construct(
        private readonly UserRepository $repository,
        private readonly NotificationService $notifier,
    ) {}
}

// ❌ BAD - 旧样板代码
class UserService
{
    private $repository;
    private $notifier;

    public function __construct(UserRepository $repository, NotificationService $notifier)
    {
        $this->repository = $repository;
        $this->notifier = $notifier;
    }
}
```

### 5. 早期返回
```php
// ✅ GOOD - 早期返回减少嵌套
public function validateUser(User $user): void
{
    if (!$user->exists) {
        throw new UserNotFoundException();
    }
    
    if ($user->is_suspended) {
        throw new UserSuspendedException();
    }
    
    // 主逻辑
}

// ❌ BAD - 过多嵌套
public function validateUser(User $user): void
{
    if ($user->exists) {
        if (!$user->is_suspended) {
            // 主逻辑
        }
    }
}
```

## 📐 方法大小限制

### 理想的方法长度
```php
// ✅ 推荐长度: 10-20 行
public function activateUser(int $userId): void
{
    $user = $this->repository->findById($userId);
    
    if ($user->is_active) {
        throw new UserAlreadyActiveException();
    }

    $user->update(['is_active' => true]);
    UserActivated::dispatch($user);
    cache()->forget("user.{$userId}");
}

// 过长示例（不推荐）- 长于 50 行
public function processOrder(Order $order): void
{
    // 50 行+ 的代码
    // 做了太多事情
}
```

## 🎯 关键检查表

提交代码前自查：

- [ ] 所有文件放在正确的目录
- [ ] 类和方法命名符合规范
- [ ] 有完整的类型提示
- [ ] 方法长度合理（< 25 行）
- [ ] 没有 PHPStan 警告
- [ ] 包含对应的测试
- [ ] 代码可读性高（变量名清晰）

- [ ] 遵循单一职责原则
- [ ] PHP-CS-Fixer 已运行
