### 1.1 统一代码规范
**目标**: 建立团队统一的编码标准

#### PSR-12 规范实践
```php
// ✅ 正确示范
namespace App\Services;

use Illuminate\Database\Eloquent\Collection;

final class UserService
{
    public function __construct(
        private UserRepository $repository,
    ) {}

    public function getActiveUsers(): Collection
    {
        return $this->repository
            ->active()
            ->with('profile')
            ->latest()
            ->get();
    }

    /**
     * 验证用户邮箱唯一性
     *
     * @param string $email 用户邮箱
     * @return bool
     */
    public function isEmailUnique(string $email): bool
    {
        return ! $this->repository->emailExists($email);
    }
}
```

**规范要点**:
- ✅ 类名使用 PascalCase，方法名使用 camelCase
- ✅ 始终指定返回类型
- ✅ 使用 readonly 属性减少样板代码
- ✅ 方法不超过 20 行（单一职责）
- ✅ 使用中文注释但代码英文
- ❌ 不允许 var_dump、dd() 在生产代码中

### 2.2 Repository 模式实践
```php
// app/Repositories/UserRepository.php
<?php

namespace App\Repositories;

use App\Models\User;
use Illuminate\Database\Eloquent\Collection;

final class UserRepository
{
    /**
     * 获取所有活跃用户，包含关联数据
     */
    public function getActive(): Collection
    {
        return User::query()
            ->where('active', true)
            ->with(['profile', 'permissions'])
            ->orderByDesc('created_at')
            ->get();
    }

    /**
     * 检查邮箱是否存在
     */
    public function emailExists(string $email): bool
    {
        return User::query()
            ->where('email', $email)
            ->exists();
    }

    /**
     * 通过 ID 获取用户（优先缓存）
     */
    public function findById(int $id): ?User
    {
        return cache()->remember(
            "user.{$id}",
            now()->addHours(24),
            fn () => User::find($id)
        );
    }

    /**
     * 创建新用户
     */
    public function create(array $data): User
    {
        return User::create($data);
    }

    /**
     * 批量更新用户状态
     */
    public function updateStatus(array $userIds, string $status): int
    {
        return User::whereIn('id', $userIds)
            ->update(['status' => $status]);
    }
}
```

### 2.3 Service 层模式
```php
// app/Services/UserService.php
<?php

namespace App\Services;

use App\Repositories\UserRepository;
use App\Events\UserActivated;
use Illuminate\Database\Eloquent\Collection;

final class UserService
{
    public function __construct(
        private UserRepository $repository,
    ) {}

    /**
     * 激活用户并触发事件
     */
    public function activateUser(int $userId): void
    {
        $user = $this->repository->findById($userId);
        
        if ($user->is_active) {
            throw new \InvalidArgumentException('用户已激活');
        }

        $user->update(['is_active' => true]);
        
        // 触发域事件，驱动其他流程
        UserActivated::dispatch($user);
        
        // 清除缓存
        cache()->forget("user.{$userId}");
    }

    /**
     * 批量发送欢迎邮件
     */
    public function sendWelcomeEmailsToNewUsers(): int
    {
        $users = $this->repository->getNewUsers();
        
        foreach ($users as $user) {
            \Mail::queue(new WelcomeEmail($user));
        }

        return $users->count();
    }
}
```

### 2.4 Action 模式（复杂操作）
```php
// app/Actions/RegisterUserAction.php
<?php

namespace App\Actions;

use App\Models\User;
use App\Services\NotificationService;
use Illuminate\Support\Facades\Hash;

final class RegisterUserAction
{
    public function __construct(
        private NotificationService $notifier,
    ) {}

    public function execute(array $data): User
    {
        // 1. 创建用户
        $user = User::create([
            'name' => $data['name'],
            'email' => $data['email'],
            'password' => Hash::make($data['password']),
        ]);

        // 2. 创建用户档案
        $user->profile()->create([
            'bio' => '',
            'avatar' => null,
        ]);

        // 3. 分配默认角色
        $user->assignRole('user');

        // 4. 发送欢迎邮件
        $this->notifier->sendWelcomeEmail($user);

        return $user;
    }
}
```

## ⚡ 第三阶段：性能优化与高可用（2-3周）

### 3.1 数据库查询优化

#### N+1 问题识别与解决
```php
// ❌ 问题代码：N+1 查询
$users = User::all();
foreach ($users as $user) {
    echo $user->profile->bio;  // 每个用户额外查询 1 次
}

// ✅ 优化后：只 2 次查询
$users = User::with('profile')->get();

// ✅ 更好：延迟加载 + 预加载
$users = User::query()
    ->with(['profile', 'permissions', 'roles'])
    ->get();
```

#### 索引策略
```php
// database/migrations/2026_03_31_create_users_table.php
Schema::create('users', function (Blueprint $table) {
    $table->id();
    $table->string('email')->unique()->index();
    $table->string('name')->index();
    $table->boolean('is_active')->index();
    $table->timestamp('created_at')->index();
    $table->timestamps();
    
    // 组合索引：常见查询条件
    $table->index(['is_active', 'created_at']);
});
```

### 3.2 缓存策略
```php
// app/Services/CacheService.php
final class CacheService
{
    /**
     * 缓存配置
     */
    private const CACHE_TTL = [
        'user' => 24 * 60,           // 用户数据: 24小时
        'config' => 7 * 24 * 60,     // 配置: 7天
        'listing' => 60,              // 列表: 1小时
    ];

    /**
     * 获取用户列表（带缓存）
     */
    public function getUserList(int $page = 1): array
    {
        $key = "users.list.{$page}";

        return cache()->remember(
            $key,
            now()->addMinutes(self::CACHE_TTL['listing']),
            fn () => User::query()
                ->paginate(15)
                ->toArray()
        );
    }

    /**
     * 清除用户相关缓存
     */
    public function clearUserCache(int $userId): void
    {
        cache()->forget("user.{$userId}");
        cache()->tags(['user'])->flush();
    }
}
```

### 3.3 队列与异步处理
```php
// app/Jobs/SendEmailJob.php
<?php

namespace App\Jobs;

use App\Models\User;
use App\Mail\NotificationMail;
use Illuminate\Bus\Queueable;
use Illuminate\Contracts\Queue\ShouldQueue;

final class SendEmailJob implements ShouldQueue
{
    use Queueable;

    public function __construct(
        private User $user,
        private string $subject,
    ) {}

    public function handle(): void
    {
        \Mail::to($this->user->email)
            ->send(new NotificationMail($this->subject));
    }

    public function failed(\Throwable $exception): void
    {
        // 记录失败日志
        \Log::error("邮件发送失败: {$exception->getMessage()}", [
            'user_id' => $this->user->id,
            'subject' => $this->subject,
        ]);
    }
}

// 使用示例
SendEmailJob::dispatch($user, '欢迎邮件')
    ->delay(now()->addMinutes(5))
    ->onQueue('emails');
```

```php
    }
```
